# 🍎 Zero-Waste Pantry (ZWP)
**Real-time, geospatial food redistribution for large-scale surplus.**

---

## 📌 Project Overview
**Zero-Waste Pantry** is a logistics platform designed to rescue high-volume surplus food from weddings, hostel messes, and religious gatherings (bhandaras). We bridge the gap between donors and collectors through a **crowdsourced distribution model** involving both registered NGOs and verified individual volunteers.

### 🚀 Core Features
* **Dual-Tier Notifications:** Immediate high-priority alerts for NGOs and a public map for volunteers.
* **Mutual Exclusion Locking:** A "Claim" system that prevents multiple people from arriving for the same food listing.
* **Real-Time Expiry:** Dynamic countdown timers to ensure food safety.
